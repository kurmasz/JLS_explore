Outputs

Create an output from a state by moving the cursor over the state, right clicking and then selecting the edit outputs.

Enter an output signal name, set the number of bits in the signal, set the output value, then click on add new output. The output will then appear in the upper window of the dialog, with the number of bits shown in brackets.

To remove an output signal, click on it in the list of outputs, then click on delete selected output.

Click on close window to make the list of outputs effective.

To view (but not edit) the outputs from a given state, move the cursor over the state, right click and then select view outputs.
