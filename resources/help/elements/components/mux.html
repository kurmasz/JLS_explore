Creating a Multiplexor

Set the number of inputs to the multiplexor by entering a number or using the keypad. The number of bits of the selector input will be automatically set to the ceiling of the log2 of this number (e.g., 6 inputs will result in a 3-bit selector input). The multiplexor will output 0 when the selector is greater than the highest numbered input.

Set the number of bits on each input by entering a number or using the keypad.

Click on OK to place the element, or Cancel to cancel its creation.
