Duplicating a Single Wire

A single input wire can be expanded to a multiple wire output bundle with each output wire having the same value as the input wire with this element. This can be useful for sign extension or sending the same control signal to one side of multiple gates.

The input to this element must be a single wire.

To create this element, click on the in the toolbar. Pick the number of output wires by typing the number in the Outputs field or via the keypad.

Pick an orientation for the element.

Click on OK to begin placing the element, or on Cancel to cancel it.
